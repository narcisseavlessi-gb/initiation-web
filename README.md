
## Contenu du Site

Le site comprend plusieurs sections :

1. **En-tête** : Logo, titre et sous-titre du projet
2. **Présentation** : Description détaillée du projet et objectifs pédagogiques
3. **Ressources** : Lien vers la documentation MDN pour approfondir
4. **Technologies** : Présentation des technologies abordées (HTML5, CSS3, bonnes pratiques)
5. **Tableau** : Planification des activités de la semaine d'initiation
6. **Pied de page** : Informations complémentaires sur le projet et modalités de rendu

## Fonctionnalités Notables

- **Design responsive** : Adapté aux différentes tailles d'écran
- **Navigation claire** : Structure sémantique facilitant la lecture
- **Tableau HTML complexe** : Utilisation de `colspan` et `rowspan` pour l'organisation des données
- **Liens externes sécurisés** : Attribut `rel="noopener noreferrer"` pour les liens hors site
- **Accessibilité** : Balises sémantiques et attributs `alt` pour les images

## Contexte Pédagogique

Ce projet s'inscrit dans le cursus de la filière MIA pour l'année universitaire 2025-2026. Il constitue une première expérience concrète en développement web avant l'abord de technologies plus avancées.
