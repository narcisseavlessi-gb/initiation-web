Initiation aux technologies du Web

Projet pédagogique de la filière MIA

Présentation du projet

Cette initiative a été mise en place pour familiariser les étudiants de la filière Mathématiques et Informatique Appliquées (MIA) avec les fondamentaux du développement web. Dans un monde où le numérique prend une place prépondérante, maîtriser les technologies web devient essentiel pour tout informaticien.

Le projet "Initiation aux technologies du Web" vise à fournir aux étudiants une première expérience concrète dans la création de pages web, en combinant théorie et pratique. Nous abordons les langages de base que sont HTML5 pour la structure et CSS3 pour la présentation.

Objectifs pédagogiques

  • Comprendre l'architecture d'une page web moderne
  • Maîtriser la sémantique HTML5
  • Appliquer les styles CSS pour une mise en page responsive
  • Structurer un projet web selon les bonnes pratiques
  • Préparer le terrain pour l'apprentissage d'autres technologies

Cette initiation sert de base solide pour les modules avancés qui suivront, notamment ceux concernant les interfaces dynamiques et les applications web complètes.
